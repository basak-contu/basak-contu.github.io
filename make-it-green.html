Make It Green
  Make It Green is a simulation game created in Webgl2 which focuses on educating people about growing the suitable tree type for given circumstances. In this way, Make It Green will make the user be more conscious about choosing the types of trees they mostly desire. Make It Green’s intention is to make people be more knowledgeable and more influential about solving the deforestation, landslide and overall environmental issues related to lack of trees.
View Demo
Project Date
January 2022
The Challenge
  It was my first experience with Three.js library. Thus, the beginning of the project was challenging until I get used to Three.js. But the most challenging part was creating different shaders written in GLSL (OpenGL Shading Language).
Project Role
It was a two person project. I had done 3D design, game design and develop.
